# 👥 Human Resources Dashboard (Tableau)

## 📌 Project Overview

This project presents an interactive Human Resources dashboard built using Tableau to analyze workforce data, employee trends, and organizational metrics. The dashboard helps in understanding employee distribution, attrition patterns, and overall HR performance.

## 🎯 Key Objectives

* Monitor employee distribution across departments and roles
* Analyze attrition and retention trends
* Evaluate workforce demographics
* Support HR decision-making with data insights

## 📊 Dashboard Features

* 👨‍💼 **Employee Overview**

  * Total Employees
  * Active vs Attrition count

* 🏢 **Department-wise Analysis**

  * Workforce distribution across departments

* 📉 **Attrition Analysis**

  * Attrition rate by department, role, and other factors

* 🎓 **Education & Job Role Insights**

  * Employee segmentation based on education and roles

* 📍 **Demographic Breakdown**

  * Gender distribution
  * Age group analysis

* 💰 **Salary Insights**

  * Salary distribution across roles and departments

* 📅 **Experience & Tenure Analysis**

  * Years at company
  * Experience trends

## 🎨 Dashboard Design

* Clean and structured layout for easy navigation
* Interactive filters for deep analysis
* Visually balanced charts for quick insights

## 💡 Business Use Cases

* Identify high attrition areas and take corrective actions
* Improve employee retention strategies
* Understand workforce composition
* Support HR planning and hiring decisions

## 🛠️ Tools & Technologies

* Tableau
* Data Visualization
* Data Analysis

## 🚀 Conclusion

This dashboard demonstrates how HR data can be transformed into meaningful insights using Tableau, enabling organizations to make informed decisions regarding workforce management and employee engagement.
